Proyectos Destacados

1. Sistema de Registro de Fallas en Producción

Este proyecto consistió en desarrollar un sistema integral de registro de fallas utilizando ASP.NET Web Forms y C# como tecnología base, y SQL Server para la gestión de la base de datos. El objetivo principal era proporcionar una solución eficiente para registrar y gestionar los tickets de fallas que surgían durante la producción.

Características del sistema:

  • Registro de Tickets: Permite a los usuarios registrar fallas indicando la fecha y hora de apertura del ticket.
  • Seguimiento de Fallas: Monitorea el tiempo de resolución registrando la fecha y hora de cierre del ticket.
  • Reporte de Tiempos de Resolución: Calcula automáticamente los minutos transcurridos entre la apertura y el cierre de los tickets, proporcionando informes detallados sobre el tiempo promedio de atención.
  • Análisis de Fallas: Genera reportes sobre las fallas más comunes y las soluciones implementadas, facilitando la identificación de problemas recurrentes.
  • Mantenimiento Programado: Incluye la funcionalidad para registrar paros programados destinados al mantenimiento o reparación de maquinaria, ayudando a planificar y minimizar el impacto en la producción.
  • Exportación de Reportes: Los informes generados se pueden exportar a formatos PDF y Excel, facilitando la distribución y análisis de los datos.

Este sistema tiene como objetivo permitir a la empresa mejorar significativamente la gestión de sus fallas, reducir el tiempo de inactividad y optimizar los procesos de mantenimiento preventivo.

2. Sistema de Consulta de Aleaciones y Elementos Químicos

El segundo proyecto se centró en la creación de una página web que funcionara como un sistema de consulta de aleaciones y sus elementos químicos, desarrollado también con ASP.NET Web Forms y C# y utilizando SQL Server para el almacenamiento de datos.

Características del sistema:

  • Consulta Dinámica: Proporciona una interfaz amigable para consultar datos de aleaciones y los elementos químicos utilizados.
  • Filtros Avanzados: Permite a los usuarios filtrar la información por fechas, tipos de hornos y elementos químicos específicos.
  • Visualización de Datos: Muestra resultados detallados incluyendo las cantidades exactas de cada elemento químico utilizado en las aleaciones.
  • Exportación de Resultados: Ofrece la opción de exportar los resultados de las consultas a formatos PDF y Excel para su análisis posterior.

Este sistema se prevee como una herramienta invaluable para los ingenieros y técnicos de la empresa, permitiéndoles acceder rápidamente a información crítica sobre las aleaciones y optimizar la producción de acuerdo con los requisitos químicos precisos.
